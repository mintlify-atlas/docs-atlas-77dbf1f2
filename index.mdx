PX4 NuttX is a standards-compliant, deeply embedded real-time operating system built on Apache NuttX, extended with PX4-specific patches that target drone autopilot hardware including Pixhawk flight controllers, STM32F4/F7/H7, i.MX RT, and Kinetis microcontrollers. It provides a POSIX/ANSI-compatible OS environment with a highly configurable build system, comprehensive driver support, and deterministic real-time scheduling — all within a tiny footprint suitable for resource-constrained MCUs.

## Get started quickly

<Steps>
  <Step title="Install build dependencies">
    Install the toolchain and host tools required for your platform (Linux, macOS, or Windows/Cygwin).

    ```bash
    sudo apt-get install gcc-arm-none-eabi make gperf flex bison libncurses-dev
    ```
  </Step>
  <Step title="Clone and configure">
    Clone the PX4 NuttX repository and configure for your target board using the configure script.

    ```bash
    git clone https://github.com/PX4/NuttX.git nuttx
    cd nuttx
    tools/configure.sh -l px4fmu-v5:nsh
    ```
  </Step>
  <Step title="Build and flash">
    Build the firmware and flash it to your hardware.

    ```bash
    make
    # Flash using OpenOCD, J-Link, or your board's bootloader
    ```
  </Step>
  <Step title="Connect to NuttShell">
    Open a serial terminal to the target's console and interact with NSH (NuttShell).

    ```bash
    # Connect at 57600 baud (or board default)
    nsh> uname -a
    NuttX 10.x.x ... arm ...
    ```
  </Step>
</Steps>
